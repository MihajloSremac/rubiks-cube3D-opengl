Rotation:
Use Arrow Keys to rotate the cube.

Movement:
(These moves make sense only if you are looking at the green center in front and white center on top)

Right Side: R, T
Left Side: L, ;
Upper Side: U, I
Down Side: D, S
Front Side: F, G
Back Side: B, N

Other Commands:

Shift – New Scramble
Backspace – Force Solve
Spacebar – Start/Stop Timer
